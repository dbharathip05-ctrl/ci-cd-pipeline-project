# AWS DevOps EKS Project

This project serves as a template for deploying applications on AWS EKS with CI/CD pipelines.

## Directory Structure
- `infrastructure/` - Contains all the infrastructure as code (IaC) files.
- `applications/` - Holds application source code and configurations.
- `ci-cd/` - Contains CI/CD configurations and scripts.
- `docs/` - Documentation for the project.
